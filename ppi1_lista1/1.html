Diferença entre Internet e Web

Internet é a infraestrutura que conecta computadores no mundo todo. Já a Web é um serviço dentro da Internet, baseado em páginas e links.

História da Internet e Web

A Internet surgiu nos anos 60 como projeto militar (ARPANET), expandindo para uso acadêmico e comercial.

Já a Web foi criada por Tim Berners-Lee em 1989, permitindo o uso de hipertexto.

  • 1969 - ARPANET
  • 1989 - Criação da Web
  • 1993 - Primeiro navegador gráfico (Mosaic)
  • 2004 - Web 2.0
